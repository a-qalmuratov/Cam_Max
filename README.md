## 📋 Joyba Haqqında

**Cam_Max** - bul zamanago'y sun'iy intellektge tiykarlanǵan video kuzatıw sisteması. Bul sistema Telegram bot arqalı islep, IP kameralarıńızdı real waqıtta baqlaw hám o'g'ırılıq, sheklengen zonalardı buzıw hám basqa qáwipli waqıyalardı avtomatik túrde anıqlawǵa imkaniyat beredi.

## 🚀 Ornatıw (Installation)

### 1️⃣ Repozitoriyani klonlaw

```bash
git clone https://github.com/yourusername/Cam_Max.git
cd Cam_Max
```

### 2️⃣ Virtual muhit jaratıw

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3️⃣ Dependencies ornatıw

```bash
pip install -r requirements.txt
```

### 4️⃣ Environment sazlaw

```bash
# Windows
copy .env.example .env

# Linux/Mac
cp .env.example .env
```

`.env` faylın ashıp, tómendegi parametrlerdi kirgiziń:

```env
BOT_TOKEN=sizning_telegram_bot_tokeningiz
GEMINI_API_KEY=sizning_gemini_api_keyingiz
```

### 5️⃣ Telegram Bot jaratıw

1. [@BotFather](https://t.me/BotFather) ga ótiń
2. `/newbot` buyruǵın jiberiń
3. Bot atın hám username kirgiziń
4. Alınǵan tokenni `.env` faylına jazıń

### 6️⃣ Botni iske túsirıw

```bash
python bot/main.py
```

## 📖 Paydalanıw Qóllanması

### 🏁 Baslaw

1. Telegram'da botıńızǵa ótiń
2. `/start` - Botni iske túsirıw
3. Atıńız hám telefon nomerıńızdı kirgiziń
4. ✅ Siz dizimnen ótkensiz!

### 📹 Kamera Qosıw

1. **Kameralar** menýusın saylań
2. **➕ Kamera qosıw** túymesın basıń
3. Kamera maǵlıwmatların kirgiziń:
   - 📝 Kamera atı (mısalı: "Kire beris")
   - 🌐 IP adresi (mısalı: 192.168.1.100)
   - 🔌 Port (standart: 554)
   - 👤 Login
   - 🔑 Parol

### 🤖 AI Izlew

Siz ózbeksha/qoraqalpoqsha sorawlar berseńiz boladi:

- "Keshegi túnde kim kirdi?"
- "Kóp adam keldi"
- "Mashina kórset"
- "O'g'ırlıq bar ma?"
